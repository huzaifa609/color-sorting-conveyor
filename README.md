# Firmware

Place your Arduino sketch(es) here (`.ino` files).

Suggested structure:
```
firmware/
├── color_sorting_conveyor/
│   └── color_sorting_conveyor.ino
```

Include at the top of your `.ino` file a short comment block describing:
- Pin mapping (color sensor, servos, motor driver)
- Libraries required (e.g. `Servo.h`)
- Calibration notes for the TCS3200 sensor
