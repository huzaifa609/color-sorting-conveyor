Cam-Driven Color Sorting Mechanism for Automation System
📌 Overview

The Cam-Driven Color Sorting Mechanism is an automated mechatronics project that sorts red and green objects using a conveyor belt, computer-controlled color detection, and a custom-designed cam-follower mechanism. The system combines mechanical design, CAD modeling, kinematic analysis, electronics, and embedded programming to demonstrate an efficient and cost-effective industrial automation solution.

🚀 Key Features
Automatic color detection using the TCS3200 RGB sensor
Arduino-based real-time control system
Custom SHM cam-follower mechanism for smooth linear motion
Conveyor belt-based automated material handling
SolidWorks 3D modeling and motion simulation
Kinematic analysis and DOF verification
Low-cost prototype fabricated using acrylic and 3D-printed components
Achieved 95% sorting accuracy with an average response time of 820 ms
🛠 Technologies Used
SolidWorks 2026

TCS3200 Color Sensor

📊 Engineering Highlights
Designed using Simple Harmonic Motion (SHM) cam profiles
Performed complete cam kinematic analysis (velocity, acceleration, pressure angle, spring stiffness)
Motion simulation validated analytical calculations within 2% error
Each cam-follower subsystem operates with 1 Degree of Freedom (DOF)
Developed a complete Bill of Materials (BOM), fabrication plan, and testing methodology
📈 Results
95% overall sorting accuracy
820 ms response time
Successfully sorted 60 test objects
Reliable continuous operation during a 1-hour endurance test
Prototype completed within a budget of PKR 17,800
